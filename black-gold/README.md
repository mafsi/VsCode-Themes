# Black & Gold Theme for VS Code

This is my first theme made for VS Code.

## Instalation

### Manual instalation

Download `black-gold-0.0.1.vsix` & install manually.
Then press `F1` in VS Code `Preferences Color theme` and select `Black Gold Theme`.
